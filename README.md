# robohackers
A repository for the 8th competition of Open Technologies in Education

Αναλυτική περιγραφή ιδέας:

Σκοπός μας είναι να κατασκευάσουμε μία μακέτα μιας έξυπνης αίθουσας του σχολείου μας, 
όπως την οραματιζόμαστε και θα θέλαμε να υπάρξει στο μέλλον. 
Η αίθουσα αυτή θα προορίζεται για δημιουργικές εργασίες και καινοτόμες δράσεις (εργασίες ομίλων δημιουργικότητας και καινοτομίας), για χαλάρωση, μελέτη ή ενασχόληση με επιτραπέζια παιχνίδια, 
καθώς επίσης να ενσωματώνει τη δανειστική βιβλιοθήκη. 
Η αίθουσα αυτή θα διαθέτει αυτοματισμούς που θα προσφέρουν ηρεμία και χαλάρωση στο χώρο, ιδιωτικότητα, συμπερίληψη και προσβασιμότητα ΑΜΕΑ, 
αλλά και σεβασμό προς το περιβάλλον εξοικονομώντας ενέργεια και περιλαμβάνοντας την ανακύκλωση.
